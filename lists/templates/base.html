Quotable

Quotable is a minimalist app for collecting and sharing quotes with friends.

I'm creating it to teach myself the basics of test-driven development, automated deployment and continuous integration as I work through Harry Percival's excellent book on these subjects.

You are welcome to use the app, of course, but keep in mind the goal of this project is to familarize myself with some new back-end tools (Django, Fabric and Jenkins specifically) and a different way of doing things - not to create a featureful app with an award-winning front-end! :)

Finally - Quotable is still a work in progress / testing. It will be a few days until all the tests are green and the features work as expected. Enjoy!
